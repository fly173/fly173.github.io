# Fly 的 GitHub Pages 博客

这是一个适合手机直接上传的静态博客模板，风格是 **科技感 + 学术风**。

## 你的目标网址
如果你的 GitHub 用户名是 `fly173`，建议仓库名创建为：

`fly173.github.io`

发布后网址通常是：

`https://fly173.github.io`

## 手机发布步骤

### 1. 在 GitHub 新建仓库
仓库名填：

`fly173.github.io`

### 2. 上传这几个文件
把压缩包解压后，上传到仓库根目录：
- `index.html`
- `style.css`

### 3. 打开 GitHub Pages
进入仓库：
- `Settings`
- `Pages`
- `Build and deployment`
- `Source` 选择 `Deploy from a branch`
- 分支选择 `main`
- 文件夹选择 `/root`
- 保存

### 4. 等几分钟
刷新后就能访问站点。
